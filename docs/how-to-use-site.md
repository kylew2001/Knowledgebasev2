# How To Use The IT Support Knowledge Base

This guide explains the main workflows for the IT Support Knowledge Base.

#TEST LINE

## Sign In

1. Open the site URL.
2. Enter your email and password.
3. Select **Sign in**.

If your account is locked after failed login attempts, ask a super admin to unlock it from the Admin Dashboard.

## User Roles

| Role | What they can do |
| --- | --- |
| Super admin | Manage users, permissions, security settings, audit logs, categories, articles, and PDFs. |
| Editor | View, create, edit, and upload knowledge base content. |
| Viewer | View published knowledge base content and PDFs only. |

## Browse The Knowledge Base

1. Select **Knowledge Base** from the side navigation.
2. Choose a category card, such as Microsoft 365, Active Directory, Networking, Printers, Endpoints, or Security.
3. Select a subcategory card to narrow the articles.
4. Open the article or PDF you need.

## Search

Use the search box at the top of the Knowledge Base page to search by:

- Article title
- Category
- Subcategory
- Tag
- Manual article text
- PDF filename or metadata

## Create A Category

Editors and super admins can create categories.

1. Open **Knowledge Base**.
2. Select **Category**.
3. Enter the category name and description.
4. Choose an icon.
5. Choose an icon colour.
6. Save the category.

Use categories for major IT support areas, such as Microsoft 365, Networking, Printers, or Security.

## Create A Subcategory

1. Open a main category.
2. Select **Add subcategory**.
3. Enter the subcategory name and description.
4. Choose an icon and colour.
5. Save the subcategory.

Examples:

- Microsoft 365 > Shared Mailboxes
- Active Directory > Password Resets
- Networking > VPN Troubleshooting
- Endpoints > New Starter Setup

## Create An Article

1. Open the correct category or subcategory.
2. Select **Add Manual/PDF Article**.
3. Enter a title.
4. Write the manual instructions in the article body.
5. Upload one or more PDFs if needed.
6. Add useful tags.
7. Save as draft or publish.

Use manual text for the steps people need quickly. Attach PDFs for vendor docs, screenshots, formal procedures, or longer reference material.

## Bulk Upload PDFs

1. Open **Knowledge Base**.
2. Select **Bulk PDFs**.
3. Choose the PDF files.
4. Select the target category and subcategory.
5. Add shared tags if needed.
6. Review the generated article titles.
7. Upload.

Each PDF should become an article record with the PDF attached. Rename unclear filenames before upload where possible.

## Super Admin Dashboard

Super admins can open **Admin Dashboard** from the side navigation.

### Users

Use the Users area to:

- Create users
- Change roles
- Disable or reactivate users
- Send password reset links
- Unlock locked accounts
- Review last login information

### Audit Logs

Use Audit Logs to review important activity, including:

- Failed logins
- Role changes
- Password reset actions
- Category changes
- Article changes
- PDF uploads and deletes
- Bulk upload activity

### Security

Use Security controls to manage:

- Failed login threshold
- Lockout duration
- Inactivity timeout

## Good Content Practices

- Use clear titles, such as **Reset a Microsoft 365 password**, not **Password thing**.
- Put the fastest fix near the top of the article.
- Add tags people are likely to search for.
- Add review dates for procedures that can go stale.
- Archive old articles instead of deleting them unless they are definitely no longer needed.
